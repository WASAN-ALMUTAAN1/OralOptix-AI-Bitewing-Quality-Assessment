# Future Work

The report proposes several directions for future improvement of OralOptix:

## 1) Expand the Dataset
Increase the dataset size and include more diverse bitewing radiographic samples to improve model robustness and generalization.

## 2) Improve Data Balance
Address class imbalance to support more stable training and better predictive performance across all classes and quality attributes.

## 3) Enhance the AI Model
Continue refining the AI model to improve accuracy and reliability in real-world use.

## 4) Improve System Integration
Further strengthen integration between the AI model and the web platform to support smoother workflows and more reliable real-time analysis and display.

## 5) Expand Practical Validation
Conduct additional testing and usability evaluation with more participants to better assess system performance and user experience in practice.
