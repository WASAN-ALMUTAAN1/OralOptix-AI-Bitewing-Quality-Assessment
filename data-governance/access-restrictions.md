# Access Restrictions

## Purpose of This Repository
This repository is published for **academic documentation**, **portfolio presentation**, and **research communication** purposes. It showcases the design, workflow, and outputs of the OralOptix project in a safe and non-sensitive format.

## What Is Included
The repository may include:
- Project documentation and structured descriptions
- User interface screenshots and workflow illustrations
- Sample AI-generated report layouts
- Presentation files and academic materials
- Non-sensitive demonstration assets

## What Is Not Included
To protect privacy, maintain ethical compliance, and respect institutional data governance requirements, this repository does **not** include:
- Real patient data
- Identifiable medical records
- Clinical images containing protected health information (PHI)
- Private institutional datasets
- Credentials, tokens, or deployment secrets
- Internal production databases or server configurations

## Restricted Components
Certain project components are intentionally excluded or abstracted:
- Full training dataset and metadata
- Raw clinical image repositories
- Production backend configuration files
- Authentication credentials and environment variables
- Internal deployment pipelines and private infrastructure details

## Access to Sensitive or Institutional Materials
If any part of the project was developed using institutional workflows, clinical collaboration, or supervised academic environments, access to underlying sensitive materials remains restricted to authorized personnel only and is governed by:
- Institutional policies
- Ethical review requirements (where applicable)
- Data protection regulations
- Supervisor or department-level approval

## Requesting Additional Information
Researchers, reviewers, or collaborators may contact the repository owner for:
- Clarification about the methodology
- High-level system design discussion
- Academic presentation materials
- Non-sensitive reproducibility notes

Please note that access requests involving protected or restricted medical data cannot be granted through this public repository.

## Repository Use Policy
By using this repository, you acknowledge that:
- The content is shared for educational and academic purposes
- Demonstration materials do not represent a deployable clinical system
- No attempt should be made to infer or reconstruct private clinical data from repository contents
